There's two way of downloading
data from the database.
  • Single parameter : Downloading a single parameter will create a file with the data of the given parameter for a given range of glaciers. This is raw data so if the parameter has replicates, all of the replicates will be in the download file.
  • Multiple parameters : Downloading multiple parameters will create a file containing merged data. The replicates for each parameter will be averaged to keep only one value and some data will be duplicate to have the same numbers of entries for each parameter so that the file contains only consistent data.
Download single parameter : To download a single parameter you need to disable multiple parameters selection. Once this is done you can select the parameter you want to download. Then select the glacier(s) you want the parameter for and finally, you can generate the download file. Now that the file has been generated you can download it on your computer by clicking the button Download file.
Download multiple merged parameters : To download a file containing multiple parameters you have to select all the desired parameters in the list and add them to the selection list on the right. You can add or remove any fields by selecting them and using the Add and Remove buttons. Then you choose your glacier(s) and you can download the generated file.
