This tab allows you to save data and files on the website. You'll save data or files depending on the data type you choose in the sidebar.

Upload data

Uploading data will generate an excel sheet like table into which you'll be able to enter your data.

Selection option: Once you chose your data type you have multiple possibilities to identify the data that you want to upload.
  • Identify your data with a single glacier ID. Recommended if your data concern only one glacier
  • Identify your data with multiple glaciers IDs. Recommended if your data concern multiple glaciers.
    • Range selection will allow you to select a range of glacier by specifying the smallest and biggest ID
    • List selection will allow you to choose exactly for whic glaciers you want to generate the table. You need to enter the IDs as a list (comma separated).
Generated table: The generated table is displayed once you clicked de "Generate" button. This table is pre-filled with mandatory fields to avoid any mistakes. The table also contains the existing data in the database. On this page, you won't be able to modify an entry with existing values but you'll only be able to upload non-existing values. All mandatory and existing entries are read-only fields.
Upload generated table: Once you've entered your data in the table you can insert it into the database by clicking on the "Upload data" button on the bottom right of the page.

Upload files

Uploading files will allow you to upload multiple files at once onto the website.

Selection option: Once you chose your data type you have multiple possibilities to identify the files that you want to upload.
  • Identify your files with a single glacier ID. Recommended if your files concern only one glacier
  • Identify your files with multiple glaciers IDs. Recommended if your files concern multiple galciers.
    • Range selection will allow you to select a range of glaciers by specifying the smallest and biggest ID
    • List selection will allow you to choose exactly for which glaciers you want to upload files. You need to enter the IDs as a list (comma separated).
Status display: Once you chose your files to upload lists of files will be displayed on the page to give you a global status of all the files in the given range. Four lists will be displayed :
  • Valid files : display the list of valid files. For a file to be valid it must have a correct filename and not already exists in the database.
  • Wrong files : display the list of wrong files. List all wrong named files.
  • Missing files : display the list of the missing files that should've been present in the given range.
  • Existing files : display the list of existing files in the database.
Upload valid files: You can then upload your files by clicking the "upload files" button. Only the valid files will be uploaded. As you can only upload files and not replace them if a file already exists in the database you won't be able to replace it.
