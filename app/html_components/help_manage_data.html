This tab allows you to manage data and files.

Manage data

Managing data will generate an excel sheet like table into which you'll be able to enter / modify data.

Selection option: Once you chose your data type you have multiple possibilities to identify the data that you want to manage.
  • Identify your data with a single glacier ID. Recommended if your data concern only one glacier. (Example : 4)
  • Identify your data with multiple glaciers IDs. Recommended if your data concern multiple glaciers.
    • Range selection will allow you to select a range of glaciers by specifying the smallest and biggest ID. (Example : 10 - 15)
    • List selection will allow you to choose exactly for which glaciers you want to generate the table. You need to enter the IDs as a list (comma separated). (Example : 4,5,6,10,12)
Generated table: The generated table is displayed once you clicked de "Generate" button. This table is pre-filled with mandatory fields to avoid any mistakes. The table also contains the existing data in the database that you can modify. All mandatory fields are read-only.
Upload generated table: Once you've updated the data in the table you can update it into the database by clicking on the "Upload data" button on the bottom right of the page.
Validation popup: When you update data, a popup will show up and display which entry will be modified in the database. This is a security so that you double-check that you don't make irreversible action by mistake and that you're sure you want to make this modification.

Manage files

Managing files will allow you to update multiple files at once onto the website.

Selection option: Once you chose your data type you have multiple possibilities to identify the files that you want to upload.
  • Identify your files with a single glacier ID. Recommended if your files concern only one glacier
  • Identify your files with multiple glaciers IDs. Recommended if your files concern multiple glaciers.
    • Range selection will allow you to select a range of glaciers by specifying the smallest and biggest ID
    • List selection will allow you to choose exactly for which glaciers you want to upload files. You need to enter the IDs as a list (comma separated).
Status display: Once you chose your files to upload lists of files will be displayed on the page to give you a global status of all the files in the given range. Four lists will be displayed :
  • Valid files : display the list of valid files. For a file to be valid it must have a correct filename
  • Wrong files : display the list of wrong files. List all wrong named files.
  • Missing files : display the list of the missing files that should've been present in the given range.
  • Existing files : display the list of existing files in the database.
Upload valid files: You can then upload your files by clicking the "upload files" button. Only the valid files will be uploaded. If a file that you upload already exists in the database then it will be overwritten by the new one.
Validation popup: When you upload files, a popup will show up and display which file will be modified in the database. This is a security so that you double-check that you don't make irreversible action by mistake and that you're sure you want to make this modification.
