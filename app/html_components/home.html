This data portal provides access to field and laboratory data collected through the Vanishing Glaciers project. The database contains geographical information regarding the sampling sites, key environmental descriptors, important microbiological measures and functional parameter. Visitors can track the status of the data generated in the different expeditions and explore this rich database using various interactive plots. Detailed information regarding field and laboratory procedures can be obtained as well.

For more information about the NOMIS project, please visit the official project website.
