Help project progress

Project progress : The project progress table displays the current state of the project. It displays what actual data is stored in the database for each expedition. When a parameter is completed for an expedition it is displayed in green with a tick.
Refresh progress : The progress table is created from the whole database and it can take long to create it so it's not updated every time you go on the web page. To update it you need to click on the Refresh button.
