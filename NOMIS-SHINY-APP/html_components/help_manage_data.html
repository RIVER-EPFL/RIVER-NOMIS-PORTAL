Manage data

Data type: You can filter the data using a date range by selecting to dates. Minimum filter resolution is daily. You can reset the date to the maximum available range with the reset button. The date range is common to all the displayed plots.
Selection option: Allows you to hide or show the sidebar contaning the plots input elements to gain more space for visualisation.
Update generated table: Allows you to hide or show the sidebar contaning the plots input elements to gain more space for visualisation.
