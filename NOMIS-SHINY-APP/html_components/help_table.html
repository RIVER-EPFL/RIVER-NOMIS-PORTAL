Date Range: You can filter the data using a date range by selecting to dates. Minimum filter resolution is daily. You can reset the date to the maximum available range with the reset button. The date range is common to all the displayed plots.
Hide/show inputs button: Allows you to hide or show the sidebar contaning the plots input elements to gain more space for visualisation.
Add unit button: Add a new visualisation unit to the page.
Remove unit button: Remove the last visualisation unit from the page.
Sidebar inputs: Use the inputs present in the sidebar to filter and plot the data that you want.
Plots: Time serie plots are brushable, meaning that you can select the date range by clicking and dragging horizontally on the plot. The filtering resolution remains daily. You can as well double click on a time serie plot to reset the date range to the maximum range available. These actions are affecting the main date range, meaning that all the displayed plot will be adjusted. In all plots, you can hover data points to get information about the point.
