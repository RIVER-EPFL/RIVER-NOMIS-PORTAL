Help creat expedition

Create a new expedition : Create an expedition will allow you to see the expedition progress in the data management tab. To create an expedition you simply need to specify these few informations :
  • Expedition's name : the name of the expedition
  • Expedition's abreviation : the abreviation given to the expedition
  • Expedition's galcier range : You need to specify all the glacier that are part of the expedition. To do so you need to specify the range of glacier corresponding to the expedition. You can add multiple range.
